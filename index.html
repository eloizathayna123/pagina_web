Exemplo de Vários Botões com addEventListener

O método addEventListener permite associar diferentes eventos a vários botões. Cada botão pode ter uma ação distinta quando clicado, como alterar o texto, mudar a cor ou exibir uma mensagem.

Escolha uma ação:

Esta é uma mensagem que mudará com os botões!

Código JavaScript:

document.getElementById('button1').addEventListener('click', function() {
  document.getElementById('textMessage').style.color = 'blue';
});
document.getElementById('button2').addEventListener('click', function() {
  document.getElementById('textMessage').textContent = 'A mensagem foi alterada!';
});
document.getElementById('button3').addEventListener('click', function() {
  document.getElementById('textMessage').style.display = 'none';
});
document.getElementById('button4').addEventListener('click', function() {
  document.getElementById('textMessage').style.display = 'block';
});
